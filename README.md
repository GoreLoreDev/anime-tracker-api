# Anime Tracker API

A RESTful backend application for tracking anime watchlists, ratings, and viewing progress, built with Java and Spring Boot.

## Project Goal

This project is being developed to learn and demonstrate backend engineering concepts including:

* REST API design
* HTTP request/response handling
* Spring Boot
* Maven
* Database integration with PostgreSQL
* JPA/Hibernate
* Authentication and authorization
* Software architecture and clean code practices

The project is being built incrementally from scratch with a focus on understanding how backend systems work rather than simply using frameworks.

---

## Planned Features

### User Management

* Create account
* Login
* User profiles

### Anime Tracking

* Add anime to personal list
* Update watch status
* Remove anime from list

### Watch Status

* Watching
* Completed
* On Hold
* Dropped
* Plan to Watch

### Ratings

* Rate anime
* Update ratings
* View rated anime

### Search

* Search anime by title
* Filter anime by status

* Java 21+
* Maven

```bash
./mvnw spring-boot:run
```

or run the main Spring Boot application class from IntelliJ IDEA.
